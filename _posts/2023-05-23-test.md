블로그를 열어 두고도 첫 글을 쓰지 않으면, 결국 기본 템플릿만 남습니다.
그래서 긴 글이 아니더라도 일단 기록을 남기기 시작하려고 합니다.

이 공간에는 완성된 지식만 올리기보다, 개발하면서 실제로 마주친 문제와 해결 과정을 정리할 예정입니다.
나중에 같은 문제를 다시 만났을 때 빠르게 되짚어 볼 수 있는 메모가 목표입니다.

앞으로는 다음 같은 내용을 자주 다루게 될 것 같습니다.

- 웹 개발 중 막힌 지점
- 설정과 배포 과정에서 놓치기 쉬운 부분
- 작더라도 반복 작업을 줄여 주는 자동화
- 다시 읽기 위한 짧은 개념 정리

시작은 단순하지만, 기록이 쌓이면 꽤 유용한 개인 문서가 될 거라고 생각합니다.

{% highlight ruby %}
def print_hi(name)
  puts "Hi, #{name}"
end
print_hi('Tom')
#=> prints 'Hi, Tom' to STDOUT.
{% endhighlight %}
